佐倉アイ ショート動画 投稿ツール

「aichan-short」は、クリエイター佐倉アイ（アイちゃん）が制作した縦型ショート動画を、 本人のTikTokアカウントへ下書き（ドラフト）としてアップロードするための個人用ツールです。 癒やし・応援をテーマにしたショート動画コンテンツを効率よく管理・投稿することを目的としています。

できること

  • 本人が制作した動画ファイルを、本人のTikTokアカウントへ下書きとしてアップロード
  • アップロードした下書きは、TikTokアプリ上で本人が最終確認して公開

このツールについて

本ツールは佐倉アイ本人およびその運用者のみが使用する個人用途のツールであり、 第三者へ機能を提供するものではありません。TikTok Content Posting API を利用しています。
